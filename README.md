### üìò Storybook

The project's components were developed using [Storybook](https://storybook.js.org/), located in the `/src/components` directory.

- The design system follows **Atomic Design** methodology (Atoms ‚Üí Molecules ‚Üí Organisms).
- Each level contains a set of components categorized by size and complexity.
- Each component includes 3 files:
  1. `index.tsx` ‚Äì the component implementation
  2. `index.stories.tsx` ‚Äì the Storybook configuration and preview
  3. `component.css` ‚Äì custom styles specific to the component
